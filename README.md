# AquaHealth-AI-Powered-Fish-Disease-Detection

## ⚙️ Project Setup & Installation

### 📌 Important Note

Make sure you are inside the correct project directory before running commands.

This project is located inside:

```
AquaHealth-AI-Powered-Fish-Disease-Detection/
```

---

### 📥 Step 1: Clone the repository

```bash id="c1"
git clone https://github.com/your-username/repo-name.git
```

---

### 📂 Step 2: Navigate to project folder

```bash id="c2"
cd AquaHealth-AI-Powered-Fish-Disease-Detection
```

> ⚠️ Do NOT run commands outside this folder.
> The `package.json` file is located inside this directory.

---

### 📦 Step 3: Install dependencies

```bash id="c3"
npm install
```

---

### 🚀 Step 4: Run development server

```bash id="c4"
npm run dev
```

---

### 🌐 Step 5: Open in browser

```
http://localhost:3000
```

---

## ❗ Common Error Fix

### ❌ Error:

```
npm ERR! missing package.json
```

### ✅ Solution:

You are in the wrong directory. Run:

```bash id="c5"
cd AquaHealth-AI-Powered-Fish-Disease-Detection
```

Then try again:

```bash id="c6"
npm install
npm run dev
```
